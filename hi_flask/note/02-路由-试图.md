
路由参数获取
写法
	<converter:variable_name>
converter类型
	string 接收任何没有斜杠（‘/’）的文件（默认）
	int	接收整型
	float	接收浮点型
	path	接收路径，可接收斜线（’/’）
	uuid	只接受uuid字符串，唯一码，一种生成规则
	any	可以同时指定多种路径，进行限定

route规则
请求方法
	@app.route(‘/rule/’,methods=[‘GET’,’POST’])
	def hello():
		return ‘LOL’
methods中指定请求方法
	GET
	POST
	HEAD 
	PUT
	DELETE
url_for
	反向解析，根据函数名字，获取反向路径
	url_for(‘函数名’,参数名=value)
