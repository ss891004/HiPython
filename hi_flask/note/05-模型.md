# 操作msyql

## 主要的几个包
    + pip install mysql-connector-python
    + pip install Flask-Migrate
    + pip install Flask-SQLAlchemy


## 运行flask 项目
    + export FLASK_APP=app
        + app 为 app.py 主入口文件

    + 数据迁移
        + export FLASK_APP=app
        + flask db init      只要一次
        + flask db migrate   迁移
        + flask db upgrade   升级， 若有model有变更， 先migrate，再upgrade。
        + flask db downgrade  降级
